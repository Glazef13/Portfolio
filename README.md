## 🛠️ Tech Stack

### Frontend Framework
- **React 19** - Modern React with latest features and hooks
- **Vite** - Lightning-fast build tool and development server

### 3D Graphics & Animation
- **Three.js** - Powerful 3D graphics library
- **React Three Fiber** - React renderer for Three.js
- **React Three Drei** - Useful helpers for React Three Fiber
- **Framer Motion** - Production-ready motion library for React
- **GSAP** - High-performance animation library

### Styling & UI
- **Tailwind CSS 4.0** - Utility-first CSS framework
- **Magic UI** - Pre-built UI components and effects
- **Aceternity UI** - Custom UI components and design system

### Additional Technologies
- **EmailJS** - Client-side email sending service
- **React Responsive** - Media queries in React
- **Maath** - Math utilities for 3D graphics
- **Cobe** - Globe visualization library

### Development Tools
- **ESLint** - Code linting and formatting
- **Git** - Version control
- **npm** - Package management

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Glazef13/Portfolio.git
cd Portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```

The application will be available at `http://localhost:5173`

4. Select "Deploy from a branch" → main → / (root)
